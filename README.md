# Moneyflow

Purchase Organizer Application for digitizing, managing, and analyzing purchase receipts.

## Project Structure

- `backend/`: FastAPI application source code.
- `frontend/`: React application (Vite + Tailwind CSS).
- `deployment/`: Scripts for deployment and containerization.
- `config.yaml`: Global system configuration.

## Technologies

- **Backend**: Python, FastAPI, SQLAlchemy, Pytesseract, Mistral AI.
- **Frontend**: React, Vite, Tailwind CSS, Zustand, Recharts.
- **Database**: PostgreSQL / SQLite.
- **Deployment**: LXC Containers.
